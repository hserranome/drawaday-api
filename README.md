
# Drawaday Consumer API

## Main tech Stack

- [NestJS](https://docs.nestjs.com/)
- [TypeORM](https://typeorm.io/#/)
- [PostgreSQL](https://www.postgresql.org/)

## Scope

The scope for this service includes the following:

- User authentication
- User data management
- Password reset

## Installation

```bash
yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn dev

# production mode
$ yarn prod
```

## Test

Uses Jest for testing.

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
